# Транслітерація · Українська → Латиниця

Онлайн-конвертер транслітерації українського алфавіту латиницею за офіційними правилами — Постанова Кабінету Міністрів України № 55 від 27 січня 2010 р. «Про впорядкування транслітерації українського алфавіту латиницею».

## Стек

- [Next.js 16](https://nextjs.org) (App Router, статичний експорт)
- [React 19](https://react.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- TypeScript

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрий [http://localhost:3000](http://localhost:3000).

## Білд

```bash
npm run build
npm run start
```

## Деплой на Vercel

Підключи репозиторій у [vercel.com/new](https://vercel.com/new) — все налаштовується автоматично. Жодних змінних середовища не потрібно.
